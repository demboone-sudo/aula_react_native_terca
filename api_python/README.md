## Como executar a api

É necessario instalar o Python a versao superior a 3.10:

Crie um ambiente virtual com o seguinte comando:

```bash
    python -m venv venv
```

Entre no diretorio e  instale as bibliotecas com o seguinte comando

```bash
    pip install -r requirements.txt
```

Agora basta executar a api, com o comando:

```bash
    python app.py
```
